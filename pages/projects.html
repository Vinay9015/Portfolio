Personalized Student Question Answering System Using LLM Prompt Engineering

Led a 3-member research team evaluating personalized LLM educational responses through systematic prompt engineering. Built a Flask web application collecting data from 37 students across 7 technical domains (Java, SQL, Data Mining, IoT, HCI, Blockchain). Tested 5 prompt personalization strategies (Default, Skills-based, Hobbies-based, Subjects-based, All-factors). Statistical analysis revealed that Skills-Based personalization achieved the best performance (lowest average rank: 2.35 vs 2.9+, highest #1 rankings: 35%). LLM responses improved understanding by an average of 1.1 points (1-5 scale), with beginners gaining the most (median +2.0 points). Developed a Python analytics pipeline for processing feedback and generating comparative visualizations.
